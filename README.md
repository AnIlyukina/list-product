### Запуск проекта

-- `npm install` - установка зависимостей

-- `npm run serve` - запуск локального сервера

-- `npm run build:prod` - сборка проекта


### Eslint
-- `npm run lint:ts` - проверка кода
-- `npm run lint:ts:fix` - автоматически исправление кода
